#  Aula 8: Projetos Front-end

## 📖 Visão Geral
A aula abordou o desenvolvimento de aplicações front-end utilizando frameworks modernos como React, Angular e Vue.js. Também foram apresentados conceitos fundamentais como configuração de ambiente com Node.js, gerenciamento de dependências com NPM, reutilização de projetos e publicação (deploy) das aplicações.

---

## 🟢 Node.js

O Node.js é um ambiente de execução JavaScript que permite rodar código fora do navegador, sendo essencial para o desenvolvimento moderno.

### 🔹 Importância:
- Permite o uso de JavaScript no backend  
- Integra desenvolvimento frontend e backend  
- Base para ferramentas e frameworks atuais  

---

## 📦 NPM (Node Package Manager)

O NPM é o gerenciador de pacotes do Node.js.

### 🔹 Funções:
- Gerenciar dependências  
- Instalar e atualizar bibliotecas  
- Organizar projetos  

### 🔹 Arquivo importante:
- `package.json` → responsável por armazenar as dependências do projeto  

---

## ⚛️ React

React é uma biblioteca JavaScript voltada para a construção de interfaces.

### 🔹 Características:
- Componentização  
- Virtual DOM (melhor performance)  
- Flexibilidade  
- Grande comunidade  

### 🔹 Estrutura:
- `src/` → código principal da aplicação  
- `public/` → arquivos estáticos  
- `App.js` → componente principal  

---

## 🅰️ Angular

Angular é um framework completo para desenvolvimento web.

### 🔹 Características:
- Uso de TypeScript  
- Arquitetura estruturada  
- Injeção de dependência  
- Data Binding  

### 🔹 Conceitos importantes:
- Componentes  
- Módulos  
- Serviços  
- Roteamento  

---

## 🟩 Vue.js

Vue é um framework progressivo utilizado para construção de interfaces.

### 🔹 Características:
- Fácil aprendizado  
- Estrutura simples  
- Componentes reutilizáveis  
- Sistema de reatividade eficiente  

### 🔹 Estrutura:
- `App.vue` → componente principal  
- `components/` → componentes reutilizáveis  
- `assets/` → arquivos estáticos  

---

## 📥 Importação de Projetos

Foi apresentado o uso de projetos prontos disponíveis na internet para agilizar o desenvolvimento.

### 🔹 Ferramentas:
- GitHub  
- Vercel  
- CodeSandbox  

### 🔹 Vantagens:
- Economia de tempo  
- Uso de boas práticas  
- Base pronta para personalização  

---

## 🚀 Atividade Proposta

- Criar projetos utilizando React, Angular e Vue  
- Versionar os projetos no GitHub  
- Importar e modificar um template  
- Realizar deploy dos projetos  
- Documentar todo o processo  

---

## 🌐 Deploy

O deploy permite disponibilizar o projeto na internet.

### 🔹 Benefícios:
- Acesso online ao sistema  
- Melhor apresentação do projeto  
- Profissionalização do desenvolvimento  

---

## 🎯 Conclusão

A aula foi essencial para compreender o ciclo completo de desenvolvimento front-end, desde a criação dos projetos até sua publicação online.

Foram reforçados conceitos importantes como:
- Uso de frameworks modernos  
- Organização e estrutura de projetos  
- Versionamento de código  
- Publicação de aplicações  

Esses conhecimentos são fundamentais para a formação na área de desenvolvimento web.
