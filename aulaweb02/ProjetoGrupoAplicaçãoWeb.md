# 📘Definição de Escopo, Prototipação e UX

## 🌐 Tipos de Aplicações Web
As aplicações web podem ser classificadas conforme seu propósito:
- Institucional  
- Blog  
- E-commerce  
- Landing Page  
- Plataforma  

---

## 📌 Escopo e Backlog do Projeto
Definir o escopo no início é essencial para:
- Alinhar expectativas com stakeholders  
- Organizar o desenvolvimento  
- Facilitar a gestão do backlog  

### Elementos importantes:
- Objetivos do projeto  
- Funcionalidades principais  
- Público-alvo  
- Tecnologias utilizadas  

### 🧾 Exemplo de escopo:
Desenvolvimento de um site institucional com:
- Página inicial atrativa  
- Página "Sobre"  
- Blog  
- Formulário de contato  
- Design responsivo  
- Uso de HTML, CSS, JavaScript e CMS  

---

## 📊 Planejamento e Gestão do Backlog
O backlog organiza as tarefas do projeto e pode ser gerenciado com:

- User Stories  
- Quadro Kanban  
- Métodos ágeis  

### 🗣️ Exemplo de User Story:
> Como visitante do site, quero acessar a página "Sobre" para conhecer melhor a empresa.

---

## 📋 Gestão de Tasks (Tarefas)
- Uso de **Kanban**  
- Organização visual das atividades  
- Acompanhamento do progresso  

---

## 🎨 Wireframes (Estrutura do Site)
Wireframe é um esboço da interface do sistema.

### Tipos:
- Baixa fidelidade (simples)  
- Alta fidelidade (detalhado)  

### Benefícios:
- Melhor organização do layout  
- Validação antes do desenvolvimento  
- Foco na usabilidade  

---

## 🛠️ Ferramentas de Prototipação
- Figma  
- Plugins (ex: html.to.design, Musho.ai)  
- Protótipos a partir de imagens e prompts  

### Objetivos:
- Validar ideias  
- Testar interfaces  
- Melhorar a experiência do usuário (UX)  

---

## 📝 Atividade Proposta
1. Pesquisar aplicações web similares  
2. Analisar estrutura e navegação  
3. Criar 3 user stories  
4. Desenvolver wireframes das telas  
5. Criar quadro Kanban (ex: Trello)  

---

## 📚 Conclusão
A aula destacou a importância de:
- Definir bem o escopo  
- Organizar tarefas com backlog  
- Utilizar wireframes e protótipos  
- Focar na experiência do usuário (UX)
