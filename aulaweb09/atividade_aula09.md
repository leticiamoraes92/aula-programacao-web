# 📝 Relatório da Atividade — API com Express

## 🎯 Objetivo
Desenvolver uma aplicação completa utilizando API REST, integrando back-end e front-end, com deploy em ambiente online.

---

## ⚙️ Parte 1 — Back-end (API)
- Desenvolvimento de uma API utilizando **Express.js**
- Implementação de rota para retorno de **data e hora**
- Testes realizados localmente para validação do funcionamento

---

## ☁️ Parte 2 — Deploy da API
- Versionamento do projeto no GitHub  
- Criação de conta na plataforma **Render**  
- Configuração e publicação como **Web Service**  
- API disponibilizada online para consumo  

---

## 💻 Parte 3 — Front-end
- Desenvolvimento de interface para consumo da API  
- Requisição dos dados (data e hora)  
- Exibição dinâmica das informações na tela
