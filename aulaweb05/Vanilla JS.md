# 💻 Aula 05 – Introdução ao JavaScript (Vanilla JS)

## 📌 Resumo

A aula apresentou os conceitos básicos para o desenvolvimento web utilizando **HTML, CSS e JavaScript**, destacando o papel de cada tecnologia na construção de páginas modernas.

---

## 🌐 HTML

- Responsável pela **estrutura da página**  
- Define elementos como:
  - 📝 Textos  
  - 🖼️ Imagens  
  - 🔗 Links  
- Utiliza **tags** para organizar o conteúdo  

---

## 🎨 CSS

- Responsável pela **aparência do site**  
- Permite:
  - 🎨 Definir cores  
  - 🔤 Alterar fontes  
  - 📐 Organizar layout  
- Pode ser aplicado de forma:
  - Inline  
  - Interna  
  - Externa (mais recomendada)  

---

## ⚙️ JavaScript

- Linguagem que adiciona **interatividade** às páginas  
- Executada no navegador (client-side)

### 🔧 Permite:
- 🖱️ Responder a ações do usuário  
- 🔄 Alterar conteúdo da página  
- 📥 Validar formulários  
- 🎯 Criar animações e efeitos  

---

## 🧩 Conceitos Importantes

### 🔹 Variáveis
- Armazenam dados  
- Podem ser declaradas com:
  - `var`
  - `let`

---

### 🔹 Condicionais
- Estruturas que tomam decisões no código  
- Exemplo: `if`, `else`

---

### 🔹 Funções
- Blocos de código reutilizáveis  
- Executam tarefas específicas  

---

### 🔹 Eventos
- Ações do usuário (ex: clique, digitação)  
- Permitem interação com a página  

---

## 🔗 Integração HTML + CSS + JS

- HTML → estrutura  
- CSS → estilo  
- JavaScript → comportamento  

👉 Juntas, formam a base do desenvolvimento web moderno.

---

## 📝 Resumo Final

A aula abordou os fundamentos de HTML, CSS e JavaScript, destacando como essas tecnologias trabalham juntas para criar páginas web interativas, dinâmicas e visualmente organizadas.
