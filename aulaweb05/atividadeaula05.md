# 💻 Aula 05 – Introdução ao JavaScript (Vanilla JS)

## 🧪 Atividade Desenvolvida

## 📌 Descrição

A atividade consistiu no desenvolvimento de uma aplicação web utilizando **HTML, CSS e JavaScript (Vanilla JS)**, aplicando conceitos de estrutura, estilização e interatividade.

🔗 Repositório do projeto:  
👉 https://github.com/leticiamoraes92/tupperware-dream-builder

---

## 🎯 Objetivo

- 🌐 Desenvolver uma aplicação web completa  
- 🎨 Aplicar estilização com CSS  
- ⚙️ Implementar interatividade com JavaScript  
- 🔄 Realizar versionamento com Git  
- 🚀 Publicar o projeto  

---

## 🛠️ Tecnologias Utilizadas

- 🌐 HTML  
- 🎨 CSS  
- 💻 JavaScript (Vanilla JS)  
- 🔗 Git e GitHub  

---

## 🧩 Estrutura do Projeto

- HTML → estrutura da página  
- CSS → estilização e layout  
- JavaScript → funcionalidades e interações  

---

## ⚙️ Funcionalidades Implementadas

- 🖱️ Interação com o usuário  
- 📄 Manipulação de elementos da página (DOM)  
- 🔄 Execução de eventos (ex: clique)  
- 📝 Possível uso de formulários  

---

## 🎨 Estilização

- Uso de CSS para:
  - 🎨 Cores e layout  
  - 📐 Organização dos elementos  
  - 📱 Melhor experiência visual  

---

## 🔄 Versionamento

- Uso do Git para controle de versões  
- Registro de alterações com commits  
- Armazenamento no GitHub
