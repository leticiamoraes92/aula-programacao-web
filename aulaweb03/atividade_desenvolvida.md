# 🐶✂️ Atividade Desenvolvida – Site de Agendamento de Banho e Tosa

## 📌 Descrição do Projeto

Foi desenvolvido um site com o objetivo de permitir o **agendamento de serviços de banho e tosa para pets**, facilitando a organização e o atendimento ao cliente.

🔗 Repositório do projeto:  
👉 https://github.com/leticiamoraes92/Site_banho_e_tosa

---

## 🎯 Objetivo

- Criar uma página web simples para agendamento  
- Aplicar conceitos de **versionamento com Git**  
- Publicar o projeto em um repositório online  
- Praticar organização e estrutura de código  

---

## 🛠️ Tecnologias Utilizadas

- 🌐 HTML  
- 🎨 CSS  
- 💻 Git  
- 🔗 GitHub
